AI-Based Cognitive Radio Spectrum Analyzer

🚀 Project Overview

The AI-Based Cognitive Radio Spectrum Analyzer is a Python-based simulation that demonstrates how intelligent systems analyze radio spectrum signals and identify available communication channels. It integrates signal processing, machine learning, and real-time visualization to mimic a cognitive radio system.

🎯 Objectives

• Simulate RF signals with noise
• Perform spectrum analysis using FFT
• Apply machine learning for power classification
• Identify and recommend free frequency channels
• Visualize spectrum activity in real time

⚙️ Key Features

• Real-time signal and noise simulation
• FFT-based spectrum analysis
• AI-based clustering using KMeans
• Automatic free channel recommendation
• Real-time spectrum visualization with GUI

🧰 Technologies Used

Python • NumPy • SciPy • Matplotlib • Scikit-learn • Tkinter
